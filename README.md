  Sistema de Controle Financeiro

Aplicação web desenvolvida com Python, Flask e SQLite para gerenciamento de receitas e despesas, permitindo o controle financeiro de forma simples e organizada.

 Funcionalidades

- Cadastro de receitas
- Cadastro de despesas
- Armazenamento de dados em banco SQLite
- Interface web utilizando Flask
- Organização dos registros financeiros
- Estrutura modular para futuras melhorias

 Tecnologias Utilizadas

- Python 3
- Flask
- SQLite
- HTML5
- CSS3

 Como Executar o Projeto

 1. Clone o repositório

```bash
git clone https://github.com/Breno1518/controle-financeirooo.git
```

 2. Acesse a pasta do projeto

```bash
cd controle-financeirooo
```

3. Crie um ambiente virtual

```bash
python -m venv venv
```

4. Ative o ambiente virtual

Windows:

```bash
venv\Scripts\activate
```

Linux/Mac:

```bash
source venv/bin/activate
```

 5. Instale as dependências

```bash
pip install -r requisitos.txt
```

6. Execute a aplicação

```bash
python app.py
```

7. Acesse no navegador

```text
http://127.0.0.1:5000
```
